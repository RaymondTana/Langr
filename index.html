How to Play Langr

Welcome to Langr - the daily language guessing game!

Objective

Guess the mystery language using the clues provided. Each day features a new language with authentic audio and text samples.

How It Works

  • You'll receive clues one-by-one to help identify the language.
  • Make your guess using the dropdown menu.
  • If incorrect, you'll get the next clue.
  • Keep guessing until you find the right answer!

Clue Order

  1. 🔊 Audio Sample - Listen to a native speaker.
  2. 👂 Phonetic Transcription - See the phonetic representation.
  3. 🔤 English Translation - Understand what was said.
  4. 🌳 Language Family - Learn about linguistic relationships.
  5. 📝 Original Text - See the text in its original script.

Good luck and have fun learning about languages from around the world! 🌍
